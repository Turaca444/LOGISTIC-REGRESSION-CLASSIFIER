# LOGISTIC-REGRESSION-CLASSIFIER
TRABAJO PRÁCTICO - SE REALIZA LO SOLICITADO: Generar 2 matriz de datos controladas de 20, 200 filas por 3 columnas - La columna 1 debe ser una distribución normal.
